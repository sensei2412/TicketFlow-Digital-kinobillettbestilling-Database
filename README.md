# TicketFlow – Digital kinobillettbestilling

TicketFlow er en enkel webapplikasjon for bestilling og administrasjon av kinobilletter.  
Applikasjonen lar brukeren fylle inn kundeinformasjon, velge billettype og antall billetter, før bestillingen lagres i en database.

Prosjektet er utviklet som en del av Oblig 3 i DATA1700 Webprogrammering.

## Funksjonalitet

- Registrering av kinobillettbestillinger
- Lagring av kundeinformasjon i database
- Oversikt over registrerte bestillinger
- Mulighet for å slette bestillinger
- Enkel frontend med HTML og JavaScript
- Backend laget med Java og Spring Boot

## Teknologier brukt

- Java
- Spring Boot
- JavaScript
- HTML
- JDBC
- H2 Database
- Maven
